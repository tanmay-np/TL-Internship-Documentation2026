🎨 Day 4: Logo Design using Inkscape

✅ Today's Focus:

  • Installed and set up Inkscape (vector graphics software)
  • Learned basic Inkscape tools: Shapes, Paths, Text, Fill, and Stroke
  • Designed a custom Tinkerers' Lab logo for my portfolio
  • Exported the logo in PNG format for web use
  • Added the new logo to my main website

🎯 Logo Design Process:

  • Research: Looked at TL branding and modern tech logos
  • Sketch: Created rough concept on paper first
  • Vector Creation: Used Inkscape's shape and path tools
  • Color Palette: Chosen blue/cyan gradient matching TL theme
  • Typography: Added "Tinkerers' Lab" text with clean font
  • Export: Saved as PNG and SVG formats

📝 Key Learnings:

  • Understanding vector graphics vs raster images
  • Using Bezier curves and paths for custom shapes
  • Working with layers to organize design elements
  • Applying gradients and transparency effects
  • Exporting graphics for web optimization
  • Importance of branding and visual identity

📝 Notes:

Today I learned Inkscape and designed my first professional logo for the Tinkerers' Lab portfolio. Vector graphics are scalable without losing quality, which is perfect for logos. I exported the logo and added it to my website's header and hero section. This logo now represents my internship branding!
